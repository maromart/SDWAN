# SD-WAN Templates
In Cisco SD-WAN, Feature Templates are attached to one-to-many Device Templates,
Device Templates are attached to one-to-many Devices


  
This script generates an spreadsheet that contains the relationship between FT-DT-Devices



## Installation
- Install requirements.txt
  - pip install -r requirements.txt
- Clone or download the repository
  - via git commands or HTTPS or SSH
- Provide the SD-WAN URL, username and password in the .env file


## Usage
Run templates.py
